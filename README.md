**Math C2** is a state-of-the-art educational application designed to make math fun, interactive, and accessible for children. Built with a focus on **privacy**, **offline performance**, and **cross-platform versatility**, Math C2 provides a gamified learning experience that scales from mobile devices to desktop computers.

## ✨ Key Features

- 🤖 **Meet Math Buddy:** An interactive mascot that guides students through lessons, provides hints, and celebrates achievements.
- 🎨 **Premium UI/UX:** A vibrant, kid-friendly interface powered by Tailwind CSS 4 and smooth animations with Framer Motion.
- 📶 **100% Offline-First:** No internet? No problem. The entire curriculum is bundled for offline use, ensuring students can learn anywhere.
- 📱 **Multi-Platform:** Available as a Desktop App (Electron), Mobile App (Android/Capacitor), and Web App (PWA).
- 🏆 **Progress Tracking:** Earn stars, unlock achievements, and receive custom-generated certificates upon topic completion.
- 🧪 **Interactive Slides:** Lessons aren't just text—they are interactive playgrounds with counters, matchers, and number lines.

## 📚 Educational Curriculum

Math C2 covers a comprehensive range of elementary math topics:

| Topic | Description | Lessons |
| :--- | :--- | :--- |
| **Numbers to 1000** | Counting, Place Value, and Comparing large numbers. | Learn + Practice |
| **Addition** | From basic sums to 3-digit addition with regrouping. | Learn + Practice |
| **Subtraction** | Mastering the art of taking away and regrouping. | Learn + Practice |
| **Multiplication** | Introduction to groups, arrays, and multiplication tables. | Learn + Practice |
| **Division** | Sharing equally and grouping basics. | Learn + Practice |
| **Fractions** | Visualizing parts of a whole with interactive pie charts. | Learn + Practice |
| **Time & Money** | Reading clocks and identifying coins/notes. | Learn + Practice |
| **Geometry** | Exploring 2D/3D shapes and their properties. | Learn + Practice |
| **Data & Patterns** | Reading bar charts, pictograms, and identifying sequences. | Learn + Practice |

## 🛠️ Technology Stack

Math C2 is built with modern, high-performance tools:

- **Frontend:** React 19 (Hooks, Router 7, Framer Motion)
- **Styling:** Tailwind CSS 4 (Next-gen CSS performance)
- **State & Data:** TypeScript for robust data modeling and local storage for progress.
- **Mobile Bridge:** Capacitor 7 for native Android integration.
- **Desktop Wrapper:** Electron for a native Windows/macOS/Linux experience.
- **Visuals:** Lucide React icons & Canvas Confetti for celebrations.
- **PWA:** Vite PWA Plugin for installation on any device.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/math-c2.git
   cd math-c2
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

---

## 📦 Building for Platforms

### Web / PWA
```bash
npm run build
# The 'dist' folder is ready for hosting or PWA installation.
```

### Desktop (Electron)
```bash
# Navigate to the electron folder
cd electron
npm install
npm run start
```

### Mobile (Android)
```bash
npx cap sync android
npx cap open android
```
