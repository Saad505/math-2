# 🚀 Step-by-Step: Pushing to GitHub

Follow these steps to upload your project to GitHub. This will also trigger the **Automatic Android APK Build**.

### Step 1: Initialize Git

Open your terminal in the project folder (`math-c2 (1)`) and run:

```bash
git init
```

### Step 2: Add and Commit Your Code

This prepares all your files for upload.

```bash
git add .
git commit -m "Finalizing Math C2: Daily Streak, Certificate, and Windows Support"
```
